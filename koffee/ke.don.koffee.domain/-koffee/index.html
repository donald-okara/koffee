Koffee

object Koffee

Central entry point for interacting with the Koffee toast notification system.

This singleton provides a simple and unified API for initializing, displaying, and managing toasts throughout your Compose application.

It handles toast host configuration, rendering customization, and lifecycle-aware toast presentation using coroutines and Compose state management.

Usage

You typically initialize Koffee once at the top level of your app and then use its show() method anywhere in your business or UI logic.

Example - Initialization

Koffee.init {
    layout { toastData -> CustomToast(toastData) }
    dismissible = true
    durationResolver = { duration ->
        when (duration) {
            ToastDuration.Short -> 2000L
            ToastDuration.Long -> 4000L
            else -> null
        }
    }
}
