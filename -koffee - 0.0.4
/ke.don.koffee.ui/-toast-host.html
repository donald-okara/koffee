ToastHost

@Composable
fun ToastHost(hostState: ToastHostState, toast: @Composable (ToastData) -> Unit, modifier: Modifier = Modifier, dismissible: Boolean = true, alignment: Alignment = Alignment.BottomCenter)

Composable function that displays toasts managed by a ToastHostState.

Toasts are displayed in a Column within a Box that fills the maximum size. Each toast can be dismissed by swiping if dismissible is true. Toasts have enter and exit animations.

Parameters

hostState

The ToastHostState that manages the toasts to be displayed.

toast

A composable lambda that defines how each ToastData should be rendered.

modifier

Optional Modifier to be applied to the toast host container.

dismissible

A boolean indicating whether toasts can be dismissed by swiping. Defaults to true.

alignment

The Alignment of the toasts within the host. Defaults to Alignment.BottomCenter. Toasts will appear from the bottom if alignment is BottomStart, BottomCenter, or BottomEnd. Otherwise, they will appear from the top.
